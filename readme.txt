Hides the Admin bar based on the user roles you have chosen.

== Description ==

Nothing could be easier, check / uncheck to display or hide the admin bar for each basic Wordpress role.

Major features in Hidden By Roles the Admin Bar include:

* Facilitates the selection of roles to hide or not the admin bar in Dashboard.

* Plugin translated into the following languages: Deutsche, Català, English, Français, Nihonjin, Português, Pǔtōnghuà.

PS: You can get the premium version managing custom roles on the site by following this address: https://kheloshop.com/boutique/wordpress/plugins/.

== Installation ==

Upload the Hidden By Roles the Admin Bar plugin to your site, activate it, and Enjoy.
